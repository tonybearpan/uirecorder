## 说明
- 该版本基于官方uirecorder
- 该版本升级chromedriver版本为125
- 该版本默认关闭校验浏览器，使用全屏窗口录制
- 报告使用界面更友好的allure
- 添加 npm run report 命令，生成并打开报告
- 添加修复样式按钮，解决样式偶尔会丢失的问题

## 安装方法

```shell
npm config set registry https://registry.npmmirror.com/
npm install uirecorder119 mocha -g
```

其他使用与官方版本无异，请参考官方文档：
